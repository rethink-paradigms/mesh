## Implementation Summary

1. **Plugin Scaffold** (plugin.go):
   - S3RegistryPlugin implements MeshPlugin interface (PluginInfo, GetAdapter)
   - RegistryConfig with env var fallbacks (AWS_REGION, AWS_ACCESS_KEY_ID, etc.)
   - S3RegistryAdapter implements SubstrateAdapter (delegates to plugin)
   - IsHealthy checks bucket existence via HeadBucket

2. **Push** (push.go):
   - Streaming multipart upload (5 MiB parts)
   - SHA-256 computed via io.TeeReader (no full buffering)
   - Metadata set via CopyObject after completion
   - Abort on error cleanup

3. **Pull** (pull.go):
   - Returns io.ReadCloser stream (no full buffering)
   - Reads SHA-256 from x-amz-meta-sha256 metadata

4. **Verify** (registry_test.go):
   - Computes SHA-256 of downloaded stream
   - Compares with metadata, returns error on mismatch

5. **Tests**:
   - Mock S3 client for isolated testing
   - Push/pull roundtrip, verify pass/fail
   - Large content (>5 MiB) multipart test
   - Error cases: empty key, missing object
   - PluginInfo and GetAdapter validation
