# Reading Maps

> What to read (files + DB queries) for each task type.
> Always read CONTEXT.md first, then use the appropriate map.

---

## Map 1: Bootstrap (Fresh Session)

For a completely fresh session with no prior context.

**Step 1**: Read current state
```
Read CONTEXT.md
```

**Step 2**: Get session context
```bash
python3 ~/.agents/skills/mesh-nav/scripts/session.py latest
```

**Step 3**: Read the project dashboard
```
Read discovery/INDEX.md
```

**Step 4**: Read intent (stable, rarely changes)
```
Read discovery/intent.md
```

**Step 5**: Check constraints
```
Read discovery/constraints.md
```

**Files read**: CONTEXT.md, SESSION-LOG.md, discovery/INDEX.md, discovery/intent.md, discovery/constraints.md
**DB queries**: session.py latest

---

## Map 2: Continuation (Resuming Work)

For sessions continuing previous work.

**Step 1**: Read current state + last session
```
Read CONTEXT.md
```
```bash
python3 ~/.agents/skills/mesh-nav/scripts/session.py latest
```

**Step 2**: Read the specific files you'll be working on
(Based on next_steps from last session)

**Step 3**: Check for drift
```bash
python3 ~/.agents/skills/mesh-nav/scripts/generate.py decisions-md > /tmp/check.md
diff discovery/state/decisions.md /tmp/check.md
```

**Files read**: CONTEXT.md, task-specific files
**DB queries**: session.py latest, drift check

---

## Map 3: Decision (Making or Reviewing Decisions)

For sessions that involve design decisions or decision review.

**Step 1**: Bootstrap (Map 1 steps)

**Step 2**: Read all decisions (generated from DB)
```
Read discovery/state/decisions.md
```

**Step 3**: Check for conflicts before deciding
```bash
python3 ~/.agents/skills/mesh-nav/scripts/gov.py conflicts <EXISTING_DECISION_ID>
```

**Step 4**: Trace relationships
```bash
python3 ~/.agents/skills/mesh-nav/scripts/gov.py trace <ID> --depth 2
python3 ~/.agents/skills/mesh-nav/scripts/gov.py enabled_by <ID>
python3 ~/.agents/skills/mesh-nav/scripts/gov.py blocked_by <ID>
```

**Step 5**: Read governance decisions
```
Read discovery/state/governance-decisions.md
```

**Step 6**: Check open questions
```bash
python3 ~/.agents/skills/mesh-nav/scripts/gov.py list --type question --status unresolved
```

**Files read**: All Map 1 files + decisions.md, governance-decisions.md
**DB queries**: gov.py conflicts, trace, enabled_by, blocked_by, list

---

## Map 4: Research (Investigating a Topic)

For research and exploration sessions.

**Step 1**: Bootstrap (Map 1 steps)

**Step 2**: Read relevant research files
```
Read discovery/research/<topic>.md
```

**Step 3**: Check personas for validation
```
Read discovery/state/personas.md
```

**Step 4**: Check constraints that apply
```
Read discovery/constraints.md
```

**Step 5**: Check related decisions
```bash
python3 ~/.agents/skills/mesh-nav/scripts/gov.py list --type decision
```

**Step 6**: Read related design docs (if applicable)
```
Read discovery/design/SYSTEM.md
Read discovery/design/deep/<topic>.md
```

**Files read**: Map 1 files + research files + personas.md + relevant design docs
**DB queries**: gov.py list
