6. Tests added:
   - TestDaemonWiresDockerAdapter: verifies docker adapter is registered and reachable
   - TestDaemonInitializesBodyManager: verifies body manager is non-nil after start
   - TestDaemonRefusesDuplicateStart: spawns child sleep process, writes its PID, verifies daemon refuses start
