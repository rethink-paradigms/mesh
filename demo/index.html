Architecture

Four Components. 530MB. That's It.

Every component was chosen for minimal memory and maximum capability. No bloat. No overhead. Just infrastructure that gets out of the way.

20 MB

Encrypted mesh VPN. Zero-config. Every node finds every other node. Agents communicate across clouds.

The Paradigm Shift

Agents Are Stateful.
Their Container Is Their Brain.

Traditional infrastructure treats containers as disposable. Kill it, restart it, who cares. But an AI agent's filesystem holds its accumulated state, context, and knowledge. Lose the container, lose the brain.

🚀

Deploy

Push code, deploy agent. One command. The agent starts with a fresh brain.

mesh agent deploy researcher --image team/researcher:v1

Evolve

Agent accumulates state in its filesystem. It learns, creates files, builds context.
