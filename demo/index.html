Architecture

Built on Battle-Tested Tools

Every component is chosen for minimal resource overhead and maximum reliability. No reinvented wheels.

P

Pulumi

Infrastructure as code with a real language. Provisions VMs across 50+ providers.

Ts

Tailscale

Zero-config WireGuard mesh networking. Encrypted tunnels between all nodes.

N

Nomad

Lightweight workload scheduler with resource-aware bin-packing. 80MB RAM.

C

Consul

Health-checked service discovery. Automatic DNS for all deployed services.

T

Traefik / Caddy

Automatic HTTPS with Let's Encrypt. Dynamic routing via Consul service mesh.

D

Docker

Standard container isolation with resource limits. Run any image from any registry.
